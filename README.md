WebClient-Kelompok2

# Deskripsi Proyek
Aplikasi ini dirancang sebagai platform berbasis web untuk menampilkan data kesehatan secara interaktif dengan antarmuka modern.  

# Fitur Utama
- **Monitoring Kesehatan**: menampilkan data seperti detak jantung, langkah, kalori, dan aktivitas harian.
- **Navigasi Multi-Halaman**: menu *Fitur, Testimoni, Tentang Kami, Mulai* dengan halaman terpisah.
- **Desain Responsif**: tampilan menyesuaikan desktop maupun mobile.
- **Integrasi GitHub**: repository sinkron dengan Visual Studio Code untuk kolaborasi tim.

---

# Teknologi yang Digunakan
- **HTML5** untuk struktur halaman.
- **CSS3** untuk styling modern (navbar, card, grid).
- **JavaScript** (opsional) untuk interaktivitas.
- **Git & GitHub** untuk version control dan kolaborasi.
